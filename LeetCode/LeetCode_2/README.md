# LeetCode Java Solutions 🚀

This repository contains high-quality Java solutions for popular LeetCode problems, organized by patterns and data structures. Each solution includes detailed comments explaining the algorithm, complexity analysis (Time/Space), and the thought process.

## 📂 Project Structure

The project is organized into packages based on the problem type:

- `src/array` - Array manipulation problems
- `src/string` - String processing algorithms
- `src/linkedlist` - Linked List operations
- `src/tree` - Binary Tree & BST algorithms
- `src/graph` - Graph traversal (DFS/BFS)
- `src/dp` - Dynamic Programming
- `src/backtracking` - Recursive backtracking solutions
- `src/util` - Helper classes (TreeNode, ListNode, ArrayUtils)
- `src/test` - Test runners for verifying solutions
